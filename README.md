## 🌟 What is AgentX?

AgentX is an intelligent, pure CLI-based coding assistant built to superpower your development workflow. It lives entirely in your terminal, meaning you don't need to switch contexts or leave your IDE's integrated terminal to get AI-powered coding help.

We support the best modern AI providers:
- **OpenAI** (GPT-4o, GPT-4)
- **Anthropic** (Claude 3.5 Sonnet, Opus)
- **Google** (Gemini 1.5 Pro)
- **XAI**
- **Mistral**
- **Groq**

## 🤔 Why use AgentX?

- **Zero Context Switching:** Stay in your terminal where you already run your git commands, builds, and tests.
- **Deep Integration:** AgentX can read your local filesystem, analyze your project context, and execute commands (with your permission).
- **Fast & Lightweight:** Built natively for Windows x64. It’s incredibly fast and doesn't hog system resources like bloated electron apps.
- **Privacy Focused:** Your code stays local until you explicitly ask the AI to analyze it.

## 🕒 When to use AgentX?

- **Refactoring:** When you need to refactor a complex file or convert a component from one framework to another.
- **Debugging:** When you hit a cryptic error in your terminal, simply ask AgentX to analyze the logs and fix the issue.
- **Scaffolding:** When starting a new project or adding boilerplate, let AgentX generate the necessary files instantly.
- **Learning:** When exploring an unfamiliar codebase, use AgentX to summarize directories and explain complex functions.

## 🚀 How to Use

### Installation

AgentX is distributed as a single unified npm package for Windows. You can install it globally on your machine:

```bash
npm install -g @agent-qofeno/agentx-cli
```

### Usage

Once installed, simply run the `agentx` command inside your project directory:

```bash
agentx
```

The CLI will launch with a beautiful interactive TUI (Terminal User Interface). 
1. Use your keyboard to navigate the interface.
2. Select your preferred AI provider and enter your API key.
3. Start chatting! You can ask AgentX to read files, run tests, or write code.
