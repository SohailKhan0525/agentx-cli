## 🌟 What is AgentX?

AgentX is an intelligent, pure CLI-based coding assistant built to superpower your development workflow. Without leaving your terminal, you can interact with state-of-the-art AI models to analyze, refactor, and generate code directly in your local environment. 

We support the best modern AI providers:
- **OpenAI**
- **Anthropic**
- **Google**

## 🚀 Quick Start

### Installation

AgentX is distributed as an npm package. You can install it globally on your machine:

```bash
npm install -g @agent-qofeno/agentx-cli
```

### Usage

Once installed, simply run the `agentx` command inside your project directory:

```bash
agentx
```

The CLI will launch with a beautiful neon-green interactive TUI (Terminal User Interface), ready to assist you.
